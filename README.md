This repository contains my complete submission for the  "Convolutional Neural Networks projects " as part of the Machine Learning / Deep Learning course at RGIPT.

Framework Used:
- PyTorch

#Datasets
- MNIST (Handwritten Digits)
- CIFAR-10 (Image Classification)

## Tasks Covered

Task 1: Environment & Data Pipeline
- GPU/CPU verification
- Dataset loading and inspection
- Data preprocessing (normalization, reshaping, one-hot encoding)
- Data augmentation pipeline

Task 2: CNN from Scratch
- Manual 2D convolution using NumPy
- Output size derivations
- Implementation of LeNet-5
- Custom CNN design for CIFAR-10

Task 3: Training & Regularisation
- Training LeNet-5 on MNIST
- Optimizer comparison (SGD, Momentum, Adam)
- Learning rate & batch size experiments
- Regularisation techniques (Dropout, BatchNorm)
- Learning rate scheduling

Task 4: Visualisation & Interpretability
- Visualization of learned filters
- Feature map analysis
- Grad-CAM implementation from scratch
- Confusion matrix & classification report

Task 5: Transfer Learning
- Feature extraction using pretrained models
- Fine-tuning strategies
- Layer unfreezing experiments
- Benchmark: scratch vs transfer learning


Key Highlights
- Implemented CNN architectures from scratch
- Achieved strong performance on MNIST and CIFAR-10
- Applied regularisation and hyperparameter tuning
- Built Grad-CAM for model interpretability
- Performed transfer learning and fine-tuning
